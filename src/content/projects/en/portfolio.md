The site you're currently browsing. The goal was simple: stay fast and pleasant to use even on an underpowered machine or a mediocre connection, while still having a real visual identity.

## Technical details

- Static rendering by default, JavaScript only where it earns its place (theme, language detection, reveal animations).
- Content (blog, projects) managed through Astro's content collections, available in both French and English.
- System fonts, native page transitions: no unnecessary dependencies.
